## Assessment Methodology

The certification assessment follows this process:

```
Phase 1: Setup (Week 1)
├── SOW execution
├── Documentation collection
└── Interview scheduling

Phase 2: Assessment (Weeks 2-4)
├── Stakeholder interviews
├── Evidence validation
└── Assessment workshops

Phase 3: Analysis (Week 5)
├── Scoring calculation
├── Gap analysis
└── Report drafting

Phase 4: Delivery (Week 6)
├── Draft review
├── Final report
└── Certification presentation
```

## Certification Criteria

### Level Achievement

| Level | Requirement |
|-------|-------------|
| **Level 1** | ≥75% of L1 criteria met |
| **Level 2** | ≥75% L1 AND ≥60% L2 criteria met |
| **Level 3** | ≥75% L1-L2 AND ≥50% L3 criteria met |

### Critical Requirements

- Critical AI Risks (EA, AGH, TM, RA): ≥50% each
- Core Practices (SR, ML): ≥40% each
- No evidence falsification
