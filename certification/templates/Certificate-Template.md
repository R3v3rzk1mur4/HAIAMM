# HAIAMM Certificate Template

## Usage Instructions

This template is for generating official HAIAMM certification certificates. Use a design tool (Figma, Canva, Adobe) to create the final visual certificate based on this content.

---

## Certificate Design Specifications

### Dimensions
- **Size:** 11" x 8.5" (Landscape) or A4 Landscape
- **Resolution:** 300 DPI for print
- **Format:** PDF (print), PNG (digital)

### Brand Colors
- **Primary:** #1E3A5F (Deep Navy)
- **Secondary:** #2E7D32 (Trust Green)
- **Accent:** #FFB300 (Gold/Achievement)
- **Background:** #FFFFFF or #F5F7FA (Light Gray)

### Fonts
- **Headers:** Montserrat Bold
- **Body:** Open Sans Regular
- **Certificate Number:** Courier New (monospace)

## Certificate Variations

### Level 1: Foundations

**Badge Color:** Bronze/Copper (#CD7F32)
**Title:** HAIAMM LEVEL 1 CERTIFIED
**Subtitle:** Foundational AI Security Governance
**Description:** demonstrating foundational maturity in Human-Assisted Intelligence security governance with basic controls implemented across core practices.

### Level 2: Certified

**Badge Color:** Silver (#C0C0C0)
**Title:** HAIAMM LEVEL 2 CERTIFIED
**Subtitle:** Managed AI Security Governance
**Description:** demonstrating managed maturity in Human-Assisted Intelligence security governance with systematic, measured controls across the organization.

### Level 3: Enterprise

**Badge Color:** Gold (#FFD700)
**Title:** HAIAMM LEVEL 3 CERTIFIED
**Subtitle:** Optimized AI Security Governance
**Description:** demonstrating optimized maturity in Human-Assisted Intelligence security governance with industry-leading practices and continuous improvement.
