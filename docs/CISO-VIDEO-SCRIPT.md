## PRODUCTION NOTES

### Visual Style Guide

**Color Palette:**
- Primary: Professional blue (#2E5090)
- Secondary: Tech green (#4CAF50)
- Accent: Warning orange (#FF9800)
- Alert: Critical red (#F44336)
- Background: Clean white/light gray

**Typography:**
- Headings: Bold, modern sans-serif (Montserrat, Roboto)
- Body: Clean, readable (Open Sans, Inter)
- Code/Technical: Monospace (Fira Code, JetBrains Mono)

**Animation Style:**
- Clean, professional motion graphics
- Not too flashy - credible and authoritative
- Data visualizations should be clear and readable
- Transitions smooth but quick (under 0.5 seconds)

### Music Recommendations

**Intro (Scenes 1-2):**
- Subtle electronic/tech undertones
- Building tension
- Example: "Tech Corporate" by Bensound

**Problem (Scenes 3-4):**
- More urgent, but not panic-inducing
- Complexity and concern
- Example: "The Elevator" by Kevin MacLeod

**Solution (Scenes 5-8):**
- Hopeful, empowering
- Modern and confident
- Example: "Inspire" by Bensound

**CTA (Scenes 9-11):**
- Inspirational, forward-looking
- Call to action energy
- Example: "Going Higher" by Bensound

### Voice Over Specifications

**Voice Type:**
- Professional, authoritative
- Mid-range pitch
- Clear enunciation
- Not overly salesy - more educational/informative

**Pacing:**
- Normal speaking pace (140-160 words per minute)
- Pause for emphasis after key statistics
- Slow down for technical terms first mention
- Speed up slightly for urgency (regulatory timeline)

**Tone:**
- Confident but not arrogant
- Urgent but not alarmist
- Solution-focused
- Credible (backed by data and real scenarios)

### B-Roll Footage Needed

**Stock Footage:**
- Modern SOC/data center
- Board meetings (diverse executives)
- Security analyst at work
- Code on screens
- Network visualizations
- Healthcare settings (for scenario)
- Fintech office (for scenario)
- Insurance company offices
- Audit/compliance documentation
- Global map (for regulatory timeline)
- GitHub interface
- Training/workshop sessions

**Custom Animation:**
- HAIAMM framework structure (3D rotating)
- 12 practices grid with icons
- Maturity level progression (0‚Üí3)
- ROI calculator
- Timeline graphics
- Network effect visualization
- Comparison tables
- Question/checklist animations

### Accessibility

**Closed Captions:**
- Full transcript for all narration
- Sound effect descriptions [dramatic music builds]
- Speaker identification if multiple voices

**Audio Description:**
- Describe key visual elements for vision-impaired
- Especially important for data visualizations

**Visual Clarity:**
- High contrast text on backgrounds
- Large enough font sizes (minimum 24pt equivalent)
- Color-blind friendly palette
- Icons paired with text labels
