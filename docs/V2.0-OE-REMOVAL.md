## Rationale for Removing OE

**Why Operations has only 3 practices:**

The Operations business function focuses on **running and maintaining** HAI systems in production. The three essential operational security practices are:

1. **Environment Hardening (EH)** - Secure the runtime environment
2. **Monitoring & Logging (ML)** - Detect and track security events
3. **Issue Management (IM)** - Respond to vulnerabilities and incidents

**What OE covered (now absorbed):**
- Secure deployment processes â†’ Covered by **EH** (Environment Hardening)
- Change management â†’ Covered by **IM** (Issue Management)
- Operational runbooks â†’ Covered by **EH** and **IM**
- Infrastructure-as-code security â†’ Covered by **EH**

**Result:** Cleaner, more focused operational security assessment without redundancy.
