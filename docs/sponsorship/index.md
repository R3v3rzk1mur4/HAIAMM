## About HAIAMM

HAIAMM is an open source framework for assessing AI-operated security programs. With 12 practices, 6 domains, and 95% OWASP alignment, it's the industry standard for AI security maturity.

- **GitHub:** [github.com/HAIAMM](https://github.com/HAIAMM)
- **Documentation:** [docs.haiamm.org](https://docs.haiamm.org)
- **Framework:** Free and open source (Apache 2.0)
